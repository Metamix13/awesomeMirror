Techniker Projekt SmartMirror

von Lukas Müller für TEUTLOFF® Technische Akademie gGmbH

Im Fokus dieser Projektarbeit soll die Entwicklung einer Frontend-Applikation für einen Smart Mirror stehen. Diese soll im Kern mit der Auszeichnungssprache HTML, im Design mit der Stylesheet Sprache CSS und in der Funktionalität in der Skriptsprache JavaScript entwickelt werden. Diese Applikation soll im Rahmen der Projektarbeit prototyphaft auf einem Raspberry-PI 3 implementiert werden. (Details siehe Projektbeschreibung)
