Technical Project SmartMirror

from Lukas Mueller for TEUTLOFF® Technische Akademie gGmbH

Content of this project is the development of an frontend-application for a smart mirror system. The core of this application is the markup language HTML with the stylesheet language CSS. The functional part is developed with Javascript/Typescript and the framework Angular 5. This application will be implemented on a Raspberry-PI 3. (For more details please have a look at the project documentation.)
